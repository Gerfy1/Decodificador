# Decodificador

Este rep contém um codificador e decodificador de texto desenvolvido como parte dos estudos de lógica e manipulação de strings 
A aplicação permite que o usuário insira um texto, converta-o para uma forma criptografada e também recupere o texto original através da função de decodificação

## Sobre o projeto

O objetivo do Decodificador é demonstrar de maneira prática como funcionam processos simples de transformação de texto.  
A aplicação foi criada para explorar:

- Manipulação de strings  
- Funções de codificação e decodificação  
- Lógica aplicada no front-end  
- Interação com o usuário por meio de interface web  

Embora utilize um método de criptografia básica o projeto é útil para compreender conceitos iniciais de algoritmos e fluxo de dados dentro de uma aplicação web.

## Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  

## Como executar o projeto

1. Baixe ou clone este repositório.  
2. Abra o arquivo `index.html` diretamente no navegador.  
3. Digite um texto na área indicada e escolha entre codificar ou decodificar.  

## Objetivo educacional

Este projeto foi desenvolvido como exercício para praticar:

- Manipulação do DOM  
- Funções e modularidade  
- Lógica de substituição de caracteres  
- Implementação de interações básicas com JavaScript
